# 🚀 Microsoft Power Apps Projects

Welcome to my Power Apps repository!

This repository contains a collection of Microsoft Power Apps projects built to solve real-world business problems using low-code development. These applications demonstrate practical use cases such as data management, workflow automation, Excel integration, and email notifications.

## 📂 Projects

### 📧 User Registration with Email Notification
A Power Apps application integrated with Power Automate that automatically sends an email notification whenever a new user submits the registration form.

**Key Features**
- User Registration Form
- Automatic Email Notifications
- Power Automate Integration
- Simple and User-Friendly Interface

---

### 📊 Excel CRUD Application
A Power Apps application connected to Microsoft Excel . Users can manage records through an intuitive interface without directly editing the Excel file.

**Key Features**
- View Records
- Add New Records
- Search Records
- Update Existing Records
- Delete Records
- Excel Integration

---
