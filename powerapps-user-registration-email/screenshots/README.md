# User Registration Notification App

## Overview

This project demonstrates how Microsoft Power Apps and Power Automate can be used together to automate user registration notifications.

Whenever a user fills out the registration form in Power Apps, Power Automate automatically sends an email notification to the administrator.

## Features

- User Registration Form
- Power Apps Canvas App
- Power Automate Cloud Flow
- Automatic Email Notifications
- Simple Low-Code Solution

## Technologies Used

- Microsoft Power Apps
- Microsoft Power Automate
- Outlook Connector

## Workflow

1. User enters registration details.
2. User clicks Submit.
3. Power Apps triggers Power Automate.
4. Email is sent automatically.
5. Administrator receives notification.
