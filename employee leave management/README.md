# Employee Leave Management System

A simple Employee Leave Management System built using Microsoft Power Platform.

## 🚀 Technologies Used

- Power Apps
- Power Automate
- SharePoint

## 📌 Current Features

- Employee can submit leave requests
- Leave requests are stored in SharePoint
- Approval workflow using Power Automate
- Leave status updates (Pending, Approved, Rejected)

## 🔄 Project Status

🚧 This project is currently under development.

New features will be added regularly as I continue learning Microsoft Power Platform.
